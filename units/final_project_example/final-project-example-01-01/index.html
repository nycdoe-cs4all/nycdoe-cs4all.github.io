Step by Step Guide

This project is an abstract composition inspired by the book Interaction of Color by Josef Albers. Each time the sketch runs, a different shade is picked for the background and the foreground rectangle, which also changes its size, within a range: the result is different each time, but the composition retains a certain character.

To generate a different result, refresh this page.
