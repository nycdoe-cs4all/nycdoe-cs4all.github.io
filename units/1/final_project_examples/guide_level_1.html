Step by Step Guide: Lines Wallpaper, Level 1

Description

This project is a pattern for a wallpaper, composed of ten lines. Each time the sketch runs, the lines have different shades, widths and x positions. The result is different each time, but the composition retains a certain character.

Learning Activities Reference

This example project applies the following concepts from previous Learning Activities:

  • Drawing basic shapes (LA 1.1)
  • Setting stroke and fill (LA 1.2)
  • Using the random function to set positions, dimensions, and colors (LA 1.2 and LA 1.3)

Steps Outline

  1. Set up the sketch
  2. Draw a random rectangle, within certain parameters

Step by Step

1) Set up the sketch

Within the setup() function, we create a canvas, 600 pixels wide and 400 pixels high : createCanvas(600, 400);

We also turn off strokes: noStroke();

The last line in our setup function is noLoop(). This tells p5 to call the draw() function only once. We will look into this in detail in the next learning activity. If you are curious, comment this line out to see what happens.

Here is the setup code in full:

function setup() {
    createCanvas(600, 400);

    //Only run the draw loop once. Try commenting this out and see what happens.
    noLoop();

    //Turn off drawing strokes
    noStroke();
}

2) Draw ten rectangles with random fills, widths, and x positions

In our draw function, we set the background to a random shade between black (0), and white (255).

Then we set the fill to another random shade of gray, and draw a rectangle. We use the random function to place the rectangle anywhere on the x axis, and to set its width to vary between 5 and 40 each time the sketch runs. The rectangle will always be drawn from the top, and be as high as the canvas. To see how we accomplish these things, take a look at the comments by the parameters we send to the rect function: 

//Set the fill to a random shade of gray
fill(random(255));

//Draw rectangle #1 
rect(random(0, width),//Set the x pos to a # between 0 and the width of the canvas
     0,               //Set the y pos to 0 (the top of the canvas)
     random(5, 40),   //Set the width of the rect to a value between 5 and 40
     height);         //Set the height of the rect to the height of the canvas

The rest of the code simply draws nine more rectangles. Each of them will have a random fill, a random x position and width, be drawn from the top, and span the height of the canvas.
